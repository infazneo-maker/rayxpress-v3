
# RayXpress V3 PRO

Commercial Delivery + Sales Tracking Platform

Includes:
- Admin Dashboard
- Restaurant Portal
- Rider Panel
- Finance Tracking
- Sales Analytics
- Order Management

## Install

1. Install NodeJS
https://nodejs.org

2. Open terminal inside this folder

Run:

npm install

Then run:

npm run dev

Open browser:

http://localhost:3000

## Deploy

1. Upload this project to GitHub
2. Connect GitHub to Vercel
3. Deploy

Your site will become live online.
